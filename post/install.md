To install `s3mon`:

    $ cargo install s3mon

> check your PATH `$HOME/.cargo/bin/s3mon`

If you don't have [`cargo`](https://doc.rust-lang.org/cargo/getting-started/installation.html) try:

```sh
$ curl https://sh.rustup.rs -sSf | sh
```
